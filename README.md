# Website Project

A basic website project repository for INST326 Lab 9. This repo is where I practice git branching, ignoring local files, and merging work through pull requests.

## Overview

This project holds the starting structure for a simple website. Work gets built on isolated feature branches and then merged into main through pull requests.

## Message Header

The message header sits at the top of every page on the site. It shows the site title and a short welcome message so visitors always know where they are. It stays the same across every page to keep the look consistent.
